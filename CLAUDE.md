# Project: 日记网页

私人日记 Web 应用，React + TypeScript + Vite + Tailwind CSS + Canvas 2D 背景动画。

## Design Context

### Users
私人日记应用，用户综合覆盖：个人情绪记录与生活回顾、创意写作与灵感捕捉、情绪健康与心理疗愈。场景多样，使用节奏私密、沉浸、个人化。

### Brand Personality
每个主题拥有独立个性，整体偏向"魔法"气质。现有 7 个主题：dreamy（梦幻渐变）、garden（晨雾花园）、berry（莓果暮光）、ocean（海盐星云）、cosmos（宇宙深空）、starlit（星烬霓虹）、nebula（星云幻境）。

### Aesthetic Direction
- 参考：游戏 UI / 奇幻主题 + 艺术展览 / 创意机构官网
- 视觉语言：华丽粒子效果、沉浸感强、像打开了另一个世界的门
- 反参考：不要极简白底、不要企业感 SaaS
- Canvas 背景动画是视觉核心，需要美丽、流动、令人沉醉

### Design Principles
1. **每个主题是独立的宇宙** — 不同主题有完全不同的视觉个性和情绪
2. **动态即体验** — 背景动画不是装饰，是内容本身；粒子、光晕、流体应有生命感
3. **层次即深度** — 多层视差、光与影的交叠才能制造沉浸感
4. **魔法第一，实用第二** — 在不影响内容可读性的前提下，优先选择令人惊叹的视觉方案
5. **每次交互都是仪式** — 悬停微动效、主题切换（粒子离散/凝聚）都应有仪式感

### Interaction Preferences
- 悬停微动效
- 主题切换动画：粒子离散旧主题，在新主题中凝聚重组
- 页面入场/退场动画
