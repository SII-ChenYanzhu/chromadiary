# Impeccable Design Context

## Design Context

### Users
私人日记应用，用户综合覆盖：个人情绪记录与生活回顾、创意写作与灵感捕捉、情绪健康与心理疗愈。场景多样，可能是夜晚一个人的安静时刻，也可能是某个灵感迸发的午后。使用节奏私密、沉浸、个人化。

### Brand Personality
每个主题拥有独立个性，整体偏向"魔法"气质：
- 梦幻渐变：米斯天 · 柔和 · 梦幻
- 晨雾花园：清新 · 宁静 · 治愈
- 莓果暮光：浪漫 · 神秘 · 沉浸
- 海盐星云：自由 · 流动 · 无边界
- 宇宙深空：魔法 · 沉浸 · 光辉
- 星烬霓虹：魔法 · 沉浸 · 光辉（电光版）
- 星云幻境：魔法 · 沉浸 · 光辉（星尘版）

### Aesthetic Direction
- **参考风格**：游戏 UI / 奇幻主题 + 艺术展览 / 创意机构官网
- **视觉语言**：华丽粒子效果、沉浸感强、像打开了另一个世界的门
- **反参考**：不要极简白底、不要企业感 SaaS、不要千篇一律的 glassmorphism 装饰
- **配色**：每个主题有专属调色板，颜色要饱满、有层次、有光感
- **动态**：Canvas 背景动画是视觉核心，需要美丽、流动、令人沉醉

### Design Principles
1. **每个主题是独立的宇宙** — 不同主题应有完全不同的视觉个性和情绪，不能像换了个颜色
2. **动态即体验** — 背景动画不是装饰，是内容本身；粒子、光晕、流体应有生命感
3. **层次即深度** — 多层视差、光与影的交叠才能制造沉浸感，不要平面
4. **魔法第一，实用第二** — 在不影响内容可读性的前提下，优先选择令人惊叹的视觉方案
5. **每次交互都是仪式** — 悬停微动效、主题切换（粒子离散/凝聚）都应有精心设计的仪式感

### Interaction Preferences
- 悬停微动效（鼠标/指尖经过时有微妙响应）
- 主题切换动画：粒子离散旧主题，在新主题中凝聚重组
- 点击波纹（可选，取决于场景）
- 页面入场/退场动画

### Technical Stack
- React + TypeScript + Vite
- Tailwind CSS
- Canvas 2D API（当前背景动画方案）
- 可升级为 WebGL / Three.js（用于更复杂的着色器效果）
